## 🔗 COMO USAR

**Para ativar um agente:**
```
/agente-[nome]

Exemplo: /agente-n8n criar workflow WhatsApp
```

**Agentes trabalham em série:**
```
/agente-mestre diagnosticar
  ↓
/agente-n8n criar arquitetura
  ↓
/agente-[específico] implementar
```
