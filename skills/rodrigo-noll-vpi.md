# 🎯 RODRIGO NOLL — MÉTODO VPI

## Propósito
Sistema completo de **Vendas Por Indicação (VPI)** — referral marketing estruturado e automatizado para crescimento escalável.

## Princípios Core
- **Indicação é o cliente mais barato** — menor CAC, maior LTV
- **Estrutura > Sorte** — programa documentado e replicável
- **Automação = Escala** — bots, funis, CRM integrados

## Frameworks Principais
1. **Estruturação do Programa VPI**
   - Definir público indicador
   - Criar incentivos (descontos, comissões, status)
   - Documentar processo

2. **Funil de Indicação**
   - Captura → Follow-up → Conversão → Recompensa

3. **Automação com n8n/CRM**
   - Rastreamento de indicadores
   - Lembretes automáticos
   - Recompensas disparadas

4. **Medição & Otimização**
   - Taxa de indicação
   - CAC vs indicação
   - ROI do programa
