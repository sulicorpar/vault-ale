Resumir esta conversa: decisões tomadas, informações novas sobre pessoas ou deals, próximas ações com dono e prazo.

Determinar onde salvar:
- Se foi sobre um deal → `deals/`
- Se foi sobre uma pessoa → `pessoas/`
- Se foi sobre conteúdo → `conteudo/`
- Se foi sobre um projeto/SaaS → `projetos/`
- Se foi sobre uma empresa → `empresas/`
- Se for genérico → `diario/`

Salvar o resumo em markdown com data no nome do arquivo.
Atualizar `memory.md` com qualquer preferência ou padrão novo.
Perguntar: "Tem mais alguma coisa para registrar antes de fechar?"
