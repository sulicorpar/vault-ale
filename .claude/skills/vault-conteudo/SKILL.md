Ler os últimos arquivos em `conteudo/` para calibrar tom e estilo.

Se o usuário trouxer uma ideia → desenvolver roteiro no formato bastidor: contexto do problema, o que está construindo, o que descobriu, próximo passo real.

Se não trouxer ideia → verificar `referencias/` e `reunioes/` para surfaçar temas quentes que estão acontecendo de verdade.

Formatos disponíveis:
- Reel curto (30-60s) — hook de bastidor + uma sacada
- Vídeo YouTube (5-15min) — tutorial ou documentário de bastidor
- Entrevista/podcast — roteiro de perguntas para convidado

Perguntar: "Para qual canal e qual formato você quer?"
