description: Começa o dia com contexto do vault — deals abertos, pessoas para contatar, prioridades.
---

Ler ou criar a nota diária de hoje em `diario/YYYY-MM-DD.md`.

Verificar `deals/` para oportunidades com próximos passos vencidos ou para hoje.
Verificar `inbox/` para arquivos não processados — perguntar se quer organizar agora.
Verificar `reunioes/` para reuniões recentes com ações pendentes.

Surfaçar as 3 principais prioridades do dia com base no que está aberto.

Perguntar: "No que vamos trabalhar hoje?"
