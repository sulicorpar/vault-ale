## 🚀 COMO USAR

### **Ativar um Agente**
```bash
/agente-[nome]

# Exemplos:
/agente-mestre diagnosticar Meta Ads
/agente-n8n criar workflow WhatsApp
/agente-debug resolver erro 404
```

### **Buscar Conhecimento**
```bash
/agente-rag buscar [tema]

# Exemplos:
/agente-rag buscar webhook n8n
/agente-rag buscar google business profile
```

### **Procurar Especialista**
```bash
/especialista [nome ou tema]

# Exemplos:
/especialista diego-ivo
/especialista seo-geo-llm
```
